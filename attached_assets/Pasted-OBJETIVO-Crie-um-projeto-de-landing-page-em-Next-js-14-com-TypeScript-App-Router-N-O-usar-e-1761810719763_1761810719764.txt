OBJETIVO
Crie um projeto de landing page em **Next.js 14+ com TypeScript** (App Router). 
NÃO usar emojis. 
Sem arquivos HTML estáticos no repo; toda a UI em TS/TSX. 
Pronto para deploy na **Netlify**.

========================
STACK E ESTRUTURA
========================
• Framework: Next.js (TypeScript), App Router.
• Diretórios:
  - app/
    - layout.tsx (Global <html> e <body> + fontes, noise overlay opcional)
    - page.tsx (landing completa)
    - globals.css (Design System e utilitários de gradiente/grão)
  - components/
    - Header.tsx
    - Hero.tsx
    - Audience.tsx
    - Outcomes.tsx
    - FormatDate.tsx
    - Professors.tsx
    - Pricing.tsx
    - LogosGrid.tsx
    - LogosMarquee.tsx
    - Notices.tsx
    - FAQ.tsx
    - Contact.tsx
    - Footer.tsx
  - lib/
    - noise.ts (gerar dataURL de textura granulada)
    - schema.ts (JSON-LD helper)
  - public/ (se precisar de assets locais)
• Config:
  - next.config.mjs com remotePatterns para imagens externas:
    hostname: "yungwizzeprod2.wordpress.com", pathname: "/wp-content/uploads/**"
  - tsconfig.json, package.json (scripts: dev, build, start)
  - netlify.toml (build = "next build"; publish = ".next"; adapter automático da Netlify para Next)
• Acessibilidade: landmarks ARIA, foco visível, prefers-reduced-motion respeitado.
